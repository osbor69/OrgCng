# orgcng

Учёт клиентов - мониторинг  

Содержит 3 процесса:  
1. Слежение за изменениями файлов и папок, указанных в базе данных  
2. Регулировка результатов голосования исполнителей за право выполнения заявки  
3. Приём и отправка сообщений из базы данных в Telegram
